1、v-bind基础用法

插值语法中可以实现对data的动态绑定，对于其它入src，href等属性我们可以使用v-bind实现。这时，可以使用v-bind 指令来动态绑定属性，v-bind用于绑定一个或多个属性值，或者向另一个组件传递 props值
